# 4. Project Planning

## Development Modules & Sequence

| # | Module | Description |
|---|---|---|
| 1 | Environment Setup & Dependency Configuration | Python venv, install Flask, scikit-learn, pandas, NumPy, Matplotlib, Seaborn |
| 2 | Machine Learning Model Selection & Architecture | Preprocess dataset, train & compare KNN, Logistic Regression, Decision Tree, Random Forest |
| 3 | Core Backend Development | Flask routes, prediction logic, input validation |
| 4 | Data Processing & Model Persistence | Feature scaling, save trained model as `.pkl` |
| 5 | Flask Frontend UI Development | HTML/CSS/Bootstrap input form + results page |
| 6 | Testing & Deployment | Local testing with Flask dev server, then deployment to Render |

## Extended Modules (added after core build)
| # | Module | Description |
|---|---|---|
| 7 | Fertilizer Advice & Soil Health Score | Compare user input to crop's ideal averages |
| 8 | Data Visualization Dashboard | EDA charts (NPK by crop, correlation heatmap, scatter plot) |
| 9 | Research Insights (K-Means) | Unsupervised clustering for Scenario 3 (research/policy) |
| 10 | Crop Comparison Tool | Side-by-side suitability scoring with radar chart |
| 11 | Live Weather Autofill | Geolocation + Open-Meteo API integration |
| 12 | PDF Report Download | ReportLab-based report generation |
